# **App Name**: Revenda Digital

## Core Features:

- Authentication: Secure authentication system using email and password, with role-based access control for multiple users per dealership.
- Vehicle Inventory Management: Intuitive dashboard for managing vehicle listings: photos, make, model, year, mileage, price, description, and status (available, sold, unavailable).
- Automatic Public Catalog: Automatically generated public-facing website (catalog) for each dealership with its own subdomain, displaying key info, and individual vehicle pages. Updates to inventory instantly reflect on the site.
- Social Media Content Generation: AI-powered tool to generate professional captions with a call to action for Instagram, along with visual post suggestions using pre-designed templates, streamlining content creation from vehicle data.
- Database Integration: Firebase Firestore is used as the backend database.

## Style Guidelines:

- Primary color: Deep blue (#30475E) to inspire trust and security in a professional environment.
- Background color: Light gray (#E8E8E8), very slightly desaturated from the primary, for a clean, neutral backdrop.
- Accent color: Orange (#F05454), creating strong contrast for calls to action.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, providing a modern and clean feel. 
- Simple, professional icons for vehicle details and calls to action.
- Clean, intuitive layout with clear information hierarchy, prioritizing ease of use for non-technical users.
- Subtle transitions and loading animations to enhance user experience without being distracting.
